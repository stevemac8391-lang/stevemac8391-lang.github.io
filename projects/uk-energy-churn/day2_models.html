Data Partition for training and prediction.

set.seed(123)

churn <- churn %>%
  mutate(Exited = factor(Exited))   # 0 = stayed, 1 = churned

train_index <- createDataPartition(churn$Exited, p = 0.8, list = FALSE)

train <- churn[train_index, ]
test  <- churn[-train_index, ]

The data set is spilt into two data set train and test. Train is used to train the model and test is used to predict and compare the results p=0.8 means the data is spilt into 80% for the train and 20% for test.
